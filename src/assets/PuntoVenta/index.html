Preguntas Frecuentes

¿Qué necesito para iniciar?

Lo único que necesitas es un dispositivo con conexión a internet (computadora, tablet, o celular)

Nota: Dependiendo del negocio es recomendable el uso de computadora ya que puede resultar mas facil y rapido de utilizar.

¿Es necesario el uso de lectora de código de barras?

Si bien no es necesario, es recomendado el uso de una lectora de código de barras ya que puede agilizar hasta un 80% la búsqueda de productos.

¿Es necesario el uso de impresora de tickets?

No es necesario el uso de impresora de ticket. Sin embargo puede ser de utilidad si deseas darle confianza a tus clientes.

¿Necesito una computadora de última generación para usar el sistema Punto de Venta?

No. El sistema Punto de Venta está diseñado para ser utilizado en dispositivos con bajos recursos, sin embargo, la velocidad del sistema Punto de Venta puede depender de la velocidad de tu dispositivo y de tu conexión a internet.

¿Puedo ver mi información en tiempo real en cualquier parte del mundo?

Sí. Con el sistema Punto de Venta puedes tener la confianza de estar en cualquier parte del mundo y ver en tiempo real las ventas en tu negocio. (Siempre que cuentes con una conexión a internet).

¿El sistema Punto de Venta es GRATUITO?

Sí, El sistema Punto de Venta está diseñado para ser gratuito. Sin embargo el adquirir una suscripción de paga puede darte beneficios como accesos rápidos, estadísticas etc.
